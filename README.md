## Setup

To run this project, install it locally using npm:

### Server side (GraphQL with MongoDB)

1. Start your local MongoDB instance
2. Run in a separated terminal:

   ```
   cd backend
   npm i
   npm run build
   npm start
   ```

### Client side (React)

In a separated terminal, run:

```
    cd client
    npm i
    npm start
```
